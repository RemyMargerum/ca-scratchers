How the numbers are built. "Prizes remaining" = the lottery's unclaimed-prize count; "beginning" = total prizes printed. Est. tickets left back-calculates the print run from the most-frequent prize tier (odds × that tier's count) — the same trick as the source script, using the largest tier for accuracy. EV / ticket = remaining prize value ÷ estimated tickets left. Return % = EV ÷ ticket price; 100% is break-even before tax. These are gross, theoretical figures: they ignore federal/CA tax on winnings, and annuity top prizes are counted at nominal (not present-cash) value, so they overstate true return.
